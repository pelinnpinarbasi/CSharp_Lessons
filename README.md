# C# Lessons

This repository contains all the lessons I followed to learn C# step by step. Each lesson focuses on a different fundamental topic of the language.

## Contents

- **C#.Lesson.1** – **Printing & Variables**  
  Learn how to print messages to the console and work with variables in C#.

- **C#.Lesson.2** – **Operators**  
  Covers arithmetic, comparison, and logical operators in C#.

- **C#.Lesson.3** – **User Input**  
  How to get input from the user and store it in variables.

- **C#.Lesson.4** – **If-Else Statements**  
  Conditional statements to control the flow of your programs.

- **C#.Lesson.5** – **Switch-Case**  
  Using switch-case statements for multiple conditional branches.

- **C# PowerPoint** – Slides and course notes for reference.

## Purpose

This repository documents my C# learning journey from basics to core fundamentals. It can serve as a portfolio project to show my progress and understanding of C#.
